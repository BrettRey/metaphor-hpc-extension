# Pilot Study Memo

## Title

Defecation-site selection relative to bounded paths in domestic dogs

## Contact

Brett Reynolds  
Humber Polytechnic & University of Toronto  
brett.reynolds@humber.ca

## Question

When dogs need to defecate, do they choose sites relative to path edges and junctions in systematic ways, and how do those choices change when off-path access is constrained?

## Why this is worth testing

Dogs appear to treat paths, edges, and linear spatial features as behaviorally important, but defecation-site choice relative to path structure seems under-studied. A bounded-path design offers a clean way to separate site preference from environmental constraint and to connect marking behavior with movement ecology in a simple, measurable way.

## Core idea

Compare dog behavior across standardized path conditions that vary in boundedness:
- open path
- low-bounded path
- high-bounded path

The key outcome is not simply whether a dog defecates on a path, but where it defecates relative to accessible alternatives.

## Minimal pilot

- Within-dog repeated-measures design
- 15-20 dogs
- 2-4 sessions per dog
- Same handler, leash length, path width, and walk protocol across sessions
- First defecation event only
- Video coding of the final pre-defecation search phase

## Primary outcomes

- lateral distance from path centerline
- path center vs path edge vs off-path location
- distance to nearest junction
- whether the dog attempted to step off the path before defecating

## Secondary measures

- latency to defecation
- sniffing and circling before defecation
- preceding urination
- leash tension / handler influence
- surface and weather conditions

## Practical setup

The study does not need to happen in Mississauga. The ideal version would use an existing collaborator site: a fenced outdoor test area, shelter exercise space, dog-walk lane, or field site already used for dog behavior research. The first pilot should stay methodologically simple: fixed cameras plus mapped path geometry, not drones or an elaborate sensor stack.

- access to an existing site that could support a bounded-path protocol
