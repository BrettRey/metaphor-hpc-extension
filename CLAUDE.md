# CLAUDE.md - Metaphor as Cross-Domain HPC Extension

## Project Overview

Paper arguing that HPC (Homeostatic Property Cluster) theory reframes Lakoff & Johnson's Conceptual Metaphor Theory (CMT). Two interlocking arguments:

**Argument A:** L&J's source domains are property clusters without stability mechanisms. HPC supplies the mechanisms. Boyd's theory-constitutive metaphors (TCMs) explain *why* metaphors succeed (accommodation to causal structure); Lakoff explains *how* they arise (embodiment). HPC fills the gap: what makes mappings stable vs ephemeral.

**Argument B:** If metaphor = cross-domain HPC extension on partial property overlap, then any animal with HPC-structured categories should show cross-domain recruitment. Dog TRAIL cognition provides evidence: dogs extend their functional spatial category TRAIL to novel instances (painted lines on soccer fields) sharing only visual-geometric properties.

## Working Title

"What Dogs Know About Trails: Metaphor, Mechanisms, and the Phylogeny of Cross-Domain Extension"

Alternative: "Categories Without Cognitivism: How HPC Theory Reframes Conceptual Metaphor"

## Key Terminological Conventions

- **TRAIL** = the dog's concrete ecological category (physical landscape feature with clustered properties maintained by use)
- **PATH** = Talmy/Lakoff's abstract linguistic schema (source-trajectory-goal)
- **HPC** = Homeostatic Property Cluster (Boyd)
- **CMT** = Conceptual Metaphor Theory (Lakoff & Johnson)
- **TCM** = Theory-Constitutive Metaphor (Boyd)
- **ICM** = Idealized Cognitive Model (Lakoff)
